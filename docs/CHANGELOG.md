# Changelog

## [v0.1.0] - Esqueleto jogável mínimo
- Definição da linguagem (Python) e setup recomendado.
- Estrutura inicial do projeto e geração básica de mapa procedural.
- Movimentação do jogador com prevenção de colisão em paredes.

## [v0.2.0] - Turnos, FOV e HUD
- Ampliação do mapa para dimensões generosas inspiradas em Caves of Qud.
- Implementação de campo de visão clássico com neblina de guerra persistente.
- Sistema de turnos integrado a cada entrada do jogador.
- HUD com estatísticas básicas do jogador e log de mensagens rolante.
